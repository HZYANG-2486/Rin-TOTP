# Rin-TOTP
一个以RinUI(PySide)构建的Python TOTP验证器
1. 支持桌面卡片模式
2. AES 加密存储
3. 多语言

## 用法

### 从源代码运行
1. 克隆该项目
2. 安装所需库
``` 
pip install -r requirements.txt
```
3. 运行!
``` 
python app_qml.py
```
### 从预编译程序运行
1. 在Releases中下载对应版本
2. 运行!
